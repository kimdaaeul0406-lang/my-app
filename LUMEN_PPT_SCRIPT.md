# LUMEN 프로젝트 최종 발표 자료 🌙
> "과장 없이, 오늘의 흐름을 정리하다"

## 1. 발표 개요 (Overview)
- **프로젝트명**: LUMEN (루멘)
- **한줄 소개**: 군더더기 없는 디자인으로 매일의 운세(사주, 타로, 별자리)를 제공하는 모던 웹 서비스
- **핵심 가치**: 심플함(Simplicity), 몰입감(Immersion), 개인화(Personalization)

---

## 2. 슬라이드별 구성 및 대본 (Script)

### 1️⃣ 표지 (Intro)
- **화면**: LUMEN 로고와 밤하늘 배경, "오늘, 당신의 흐름은 어디로 가고 있나요?" 문구
- **대본**: 
  "안녕하세요, **과장 없이 오늘의 흐름을 정리하는 서비스, LUMEN**을 발표하게 된 [이름]입니다. 복잡하고 광고가 많은 운세 서비스들에 지치셨나요? LUMEN은 오직 '나의 흐름'에만 집중할 수 있는 가장 심플하고 아름다운 공간을 제공합니다."

### 2️⃣ 기획 의도 (Why LUMEN?)
- **화면**: 기존 운세 사이트의 복잡한 화면(X) vs LUMEN의 깔끔한 UI(O) 비교
- **키워드**: `No Ads`, `No Login (Optional)`, `Minimalism`
- **대본**:
  "기존 운세 서비스들은 광고가 너무 많고, 디자인이 올드합니다. 사용자는 그저 '오늘 나의 운세'가 궁금할 뿐인데 말이죠.
  그래서 우리는 **불필요한 요소를 모두 제거**했습니다. 
  1. **광고 없음**: 오직 콘텐츠에만 집중
  2. **가입 강요 없음**: 접속하자마자 결과 확인
  3. **몰입감 있는 디자인**: 밤하늘을 수놓는 별들처럼 아름다운 시각적 경험을 선사합니다."

### 3️⃣ 주요 기능 (Core Features) - 🌟 업데이트!
- **화면**: 3가지 메인 기능 카드 (사주, 타로, 별자리) + **뉴스레터 구독 모달**
- **내용**:
  - **사주/별자리**: 생년월일 기반 맞춤형 분석
  - **타로**: 직접 카드를 섞고 뽑는 인터랙티브 경험 (애니메이션)
  - **[NEW] 뉴스레터 구독**: 매일 아침 이메일로 받아보는 운세
  - **[NEW] 결과 이메일 전송**: 타로 카드 이미지까지 포함된 예쁜 결과지 소장 가능
- **대본**:
  "LUMEN의 경험은 웹에서 끝나지 않습니다.
  사용자는 사주, 별자리, 타로 결과를 즉시 확인할 수 있을 뿐만 아니라, **'구독하기'**를 통해 매일 아침 뉴스레터를 받아볼 수 있습니다.
  특히 타로 결과는 **실제 뽑은 카드의 이미지와 함께** 이메일로 전송되어, 나만의 결과를 소장하는 특별한 경험을 제공합니다."

### 4️⃣ 기술 스택 (Tech Stack)
- **화면**: 로고 아이콘 나열 
  - **Frontend**: `Next.js 15` (App Router), `TypeScript`, `React`
  - **Style**: `Vanilla CSS` (최적화된 애니메이션), `Glassmorphism`
  - **Backend/DB**: `Supabase` (PostgreSQL, Realtime)
  - **Email**: `Resend API` (안정적인 이메일 발송)
  - **AI**: `Google Gemini 2.0 Flash` (고도화된 자연어 운세 해석)
  - **Deployment**: `Vercel`
- **대본**:
  "최신 기술 스택을 활용해 성능과 사용자 경험을 모두 잡았습니다.
  **Next.js 15**의 App Router로 빠른 로딩 속도를 구현했고, **Supabase**로 데이터를 안전하게 관리합니다.
  운세 해석에는 **Gemini 2.0 Flash AI**를 도입하여, 단순한 DB 조회가 아닌 맥락에 맞는 풍부한 해석을 생성합니다.
  또한 **Resend API**를 연동하여 아름다운 HTML 이메일을 실시간으로 전송합니다."

### 5️⃣ 시연 (Demo Flow) - 🎬 가장 중요!
> **발표 시 직접 보여줄 순서입니다.**

1. **메인 페이지 진입**
   - 밤하늘 별 애니메이션과 타자기 효과 ("오늘, 당신의 흐름은...") 보여주기
2. **뉴스레터 구독 시연 (핵심 ✨)**
   - 하단 '이메일로 받아보기' 클릭
   - **모달**이 뜨고 이메일 입력
   - "구독 완료" 토스트 메시지와 함께 환영 이메일 도착 확인
3. **타로 카드 뽑기**
   - '고민 해결 타로' 선택
   - **카드 셔플 및 펼치기 애니메이션** 시연 (부드러운 인터랙션 강조)
   - 카드 선택 후 결과 확인
4. **결과 이메일 전송**
   - 결과 화면 하단 '이메일로 결과 받기' 클릭
   - 이메일 전송 후, **실제 도착한 메일 보여주기** (타로 카드 이미지가 포함된 예쁜 디자인 강조)

### 6️⃣ 마무리 (Outro)
- **화면**: "LUMEN" 로고와 함께 "지금 바로 확인해보세요"
- **대본**:
  "LUMEN은 단순한 운세 사이트가 아닙니다. 
  바쁜 현대인들이 잠시 멈춰 서서, 자신을 돌아보고 위로받을 수 있는 **디지털 힐링 공간**입니다. 
  매일 아침, LUMEN과 함께 하루의 흐름을 읽어보세요. 감사합니다."

## 3. 예상 질문 & 답변 (Q&A)

**Q. AI가 해석하면 정확도가 떨어지지 않나요?**
A. "기존의 딱딱한 DB 텍스트 조합 방식보다 훨씬 자연스럽습니다. 사주 명리학과 타로 해석 데이터를 프롬프트 엔지니어링을 통해 AI에게 학습시켰기 때문에, 전문적인 용어를 쉬운 말로 풀어주면서도 깊이 있는 해석을 제공합니다."

**Q. 이메일은 스팸으로 분류되지 않나요?**
A. "Resend라는 신뢰도 높은 이메일 API 서비스를 사용하고 있으며, 도메인 인증(DKIM/SPF)을 통해 스팸 분류 가능성을 최소화했습니다. 구글, 네이버 등 주요 메일 서비스에서 정상 수신됨을 테스트 완료했습니다."

**Q. 앱으로 출시할 계획은 없나요?**
A. "현재 PWA(Progressive Web App) 기술이 적용되어 있어, 별도의 앱 설치 없이도 '홈 화면에 추가'를 통해 앱처럼 사용할 수 있습니다."
